# 개별 리포트 목록 조회

카테고리: report
설명: 개별 리포트 목록을 조회
Method: GET
URL: /report/individual
param: page , size, sort
사용자: 유저
상태: 진행 중

### Request

**Query parameter**

| **파라미터명** | **타입** | **필수 여부** | **기본값** | **설명** |
| --- | --- | --- | --- | --- |
| `page` | Integer | Optional | `0` | 조회할 페이지 번호 (0부터 시작) |
| `size` | Integer | Optional | `10` | 한 페이지당 노출할 리포트 개수 |
| `sort` | String | Optional | `reportDate,desc` | 정렬 기준 (기본값: 최신 날짜순) |

### Response

`content[]` : 조회된 개별 리포트 목록

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `reportId` | 개별 리포트 내부 ID  | Long | 필수 | False | `12` |
| `status` | 리포트 생성 상태  | String | 필수 | False | `"COMPLETED"` (PENDING, COMPLETED, FAILED)  |
| `title` | 리포트 제목  | String | 선택  | True  | `"12번 개별 검사 요약 보고서"` |
| `createdAt` | 리포트 기준일  | String (DateTime) | 필수 | False | `"2026-07-06T18:00:00"` |
| `updatedAt` | 리포트 수정 | String (DateTime) | 필수 | False | `"2026-07-06T18:30:00"` |

`pageable` : 페이징 정보

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `page` | 현재 페이지 번호 (0부터 시작) | Integer | 필수 | False | `0` |
| `pageSize` | 한 페이지당 요청한 데이터 개수 | Integer | 필수 | False | `10` |
| `totalElements` | 조건에 맞는 전체 데이터 총 개수 | Long | 필수 | False | `45` |
| `totalPages` | 조건에 맞는 전체 페이지 수 | Integer | 필수 | False | `5` |

**Example**

```jsx
{
  "success": true,
  "message": "개별 리포트 목록 조회가 완료되었습니다.",
  "data": {
    "content": [
      {
        "reportId": 12,
        "status": "COMPLETED",
        "title": "12번 개별 검사 요약 보고서",
        "createdAt": "2026-07-06T18:00:00",
        "updatedAt": "2026-07-06T18:30:00"
      }
    ],
    "pageable": {
      "page": 0,
      "pageSize": 10,
      "totalElements": 45,
      "totalPages": 5
    }
  }
}
```

### Status

| status | response content |  |
| --- | --- | --- |
| **200**  | OK | 개별 리포트 목록이 정상적으로 조회됨  |
| **400**  | Bad Request | `page`나 `size` 쿼리 파라미터에 숫자가 아닌 잘못된 형식이 들어온 경우  |
| **401** | Unauthorized | 헤더에 JWT 토큰이 누락되었거나 만료되어 인증에 실패한 경우 |
| **500** | Internal Server Error | DB 커넥션 오류 등 서버 내부 문제로 조회가 불가능한 경우 |
