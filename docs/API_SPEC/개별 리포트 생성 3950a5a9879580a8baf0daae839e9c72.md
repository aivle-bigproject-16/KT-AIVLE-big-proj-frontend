# 개별 리포트 생성

카테고리: report
설명: 개별 리포트 생성
Method: POST
URL: /reports/individual
사용자: 유저
상태: 진행 중

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `batteryCellId` | 배터리 셀 아이디 | int |  | X | 1 |
| `forceRegenerate` | 기존 리포트가 있을 때 재생성 여부 | Boolean | true, false | O | false |

**Example**

```jsx
{
  "batteryCellId": 12,
  "forceRegenerate": false
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `reportId` | 생성된 개별 리포트 ID | Number | - | X | 1 |
| `reportDate` | 리포트 기준일 | String | yyyy-MM-dd | X | "2026-07-06" |
| `status` | 리포트 생성 상태 | String | PENDING, COMPLETED, FAILED | X | "PENDING" |
| `message` | 처리 메시지 | String | - | O | "개별 리포트 생성 요청이 접수되었습니다." |
| `createdAt` | 생성 시각 | String | ISO DateTime | X | "2026-07-06T14:30:00" |

**Example**

```jsx
{
  "success": true,
  "message": "개별 리포트 생성 요청이 완료되었습니다.",
  "data":{
	  "reportId": 1,
	  "reportDate": "2026-07-06",
	  "status": "PENDING",
	  "createdAt": "2026-07-06T14:30:00"
	}
}
```

### Status

| status | response content |  |
| --- | --- | --- |
| 201 | CREATED | 개별 리포트 생성 요청 성공 |
| 400 | INVALID_BATTERY_ID | 배터리 아이디 형식이 잘못됨 |
| 401 | UNAUTHORIZED | 인증 필요 |
| 403 | FORBIDDEN | 권한 없음 |
| 409 | INDIVIDUAL_REPORT_ALREADY_EXISTS | 해당 배터리의 리포트가 이미 존재함 |
| 500 | REPORT_GENERATION_FAILED | 리포트 생성 중 서버 오류 |
