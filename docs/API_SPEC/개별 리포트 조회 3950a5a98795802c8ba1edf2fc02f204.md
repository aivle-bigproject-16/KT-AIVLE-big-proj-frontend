# 개별 리포트 조회

카테고리: report
설명: 개별 리포트 조회
Method: GET
URL: /report/individual/:reportId
param: :reportId
사용자: 유저
상태: 진행 중

### Request

**Query parameter**

| **파라미터명** | **타입** | **필수 여부** | **기본값** | **설명** |
| --- | --- | --- | --- | --- |
| `reportId` | Number | Required | - | 생성된 개별 리포트 ID |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `batteryCellId` | 배터리 셀 id | BIGINT | - | X | 1 |
| `cellSerialNo` | 외부에서 식별하는 배터리 cell serial number | String |  | X | “CELL-001” |
| `status` | 리포트 생성 상태  | String | - | X | `"COMPLETED"` (PENDING, COMPLETED, FAILED)  |
| `title` | 리포트 제목 | VARCHAR(255) | - | O | “상세 리포트” |
| `content` | 리포트 본문 | TEXT | - | O | “CT 이미지 분석…” |
| `ctImageUrl` | 대표 ct 이미지 url | String |  | O |  |
| `rgbImageUrl` | 대표 RGB 이미지 url | String |  | O |  |
| `createdAt` | 생성일 | TIMESTAMP | - | X | "2026-07-07T10:00:00Z" |
| `updatedAt` | 수정 시각 | TIMESTAMP | - | O | "2026-07-07T10:30:00Z" |

`imageMappings[]`  : 이미지별 bbox

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `imageType` | ct 이미지 또는 rgb 이미지 | STRING |  |  | CT |
| `imageId` | ct 이미지 id 또는 rgb 이미지 id | BIGINT | - | X | 1 |

`imageMappings[].bbox`  : bbox

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `x` |  | Float |  | X | 120 |
| `y` |  | Float | - | X | 80 |
| `width` |  | Float | - | X | 240 |
| `height` |  | Float | - | X | 160 |

**Example**

```jsx
{
  "success": true,
  "message": "개별 리포트 조회가 완료되었습니다.",
  "data": {
		"batteryCellId": 1,
		"cellSerialNo": "CELL-001",
		"status": "COMPLETED",
	  "title": "BATT-001 불량 검사 상세 리포트",
	  "content": "CT 이미지 분석 결과, 상단 탭 부분에 94% 신뢰도로 크랙이 발견되었습니다.",
	  "rgbImageUrl": "",
	  "ctImageUrl": "",
	  "createdAt": "2026-07-07T10:00:00Z",
	  "updatedAt": "2026-07-07T10:30:00Z",
	  "imageMappings": [
		  {
			  "imageType": "CT",
			  "imageId": 1,
			  "bbox":{
					"x": 120,
					"y": 80,
					"width": 240,
					"height": 160
				}
			}
	  ]
	}
}
```

### Status

| status | response content |  |
| --- | --- | --- |
| 200 | OK | 개별 리포트 조회 성공 |
| 400 | INVALID_PARAMETER | 리포트 아이디 파라미터 형식이 잘못됨 |
| 404 | REPORT_NOT_FOUND | 해당 배터리의 리포트가 아직 없음 |
| 500 | INTERNAL_SERVER_ERROR | 리포트 조회 중 서버 오류 |
