# 검사 진행 상황 수신

카테고리: simulation
설명: 시뮬레이션 진행 중 촬영 진행률 갱신 시 발생
Method: WS
URL: /ws/sim
param: progress, completed
사용자: 유저
상태: 진행 중

### Response (Payload 스키마)

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| event | 이벤트 상태 구분 | enum(
PROGRESS,
COMPLETED) |  | X | PROGRESS |
| batchCount | 전체 배치 개수 | int |  | X | 6 |
| batteryCellCount | 전체 배터리 개수 | int |  | X | 12 |
| captureSpeed | 촬영 속도 | int |  | X | 30 |

`registered[]` : 대기중인 배치 리스트

`capture[]` : 촬영 중 및 촬영 완료된 배치 리스트(셀 촬영 및 촬영 완료 셀 적재 공간)

`analyze` : 분석중인 배치

`completed` : 공정 완료 배치

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| batchId | 배치 아이디 | int |  | X | 6006 |
| status | 진행 상태 | enum(
`REGISTERED` ,
`CAPTURING` ,
`CAPTURED` ,
`ANALYZING` ,
`ANALYZED` ,
`COMPLETED` ) |  | X |  |

`cells` : 셀 정보

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| batteryCellId | 셀 아이디 | int |  | X | 6001 |
| inspectionId | 검사 ID | int |  | X | 5001 |
| finalLabel | 판단 여부 | enum(
`PASS` ,
`REJECT` ,
null) |  | O |  |
