# 대시보드 기본 정보 조회

카테고리: dashbord
설명: 대시보드 기본 정보를 조회
Method: POST
URL: /dashboard
사용자: 유저
상태: 진행 중

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `todayDate` | 오늘 날짜(그래프 마지막 날짜) | STRING(DATE) |  | X | `"2026-07-07"` |
| `startDate`  | 그래프 시작 날짜 | STRING(DATE) |  | O | `"2026-07-01"` |
| `size` | 노출할 최신 검사 이력 개수 | INT |  | X | `5` |
| `graphType` | 상세 분석 그래프 타입 | ENUM(”DEFECT_TYPE”, “DAILY_TREND” , “MANUFACTURE_DEFECTS”) |  | X | `DEFECT_TYPE`  |

분석 시각화 종류(그래프): 제조사별 불량, 불량 유형별, 날짜별 추이

**Example**

```jsx
{
	"todayDate": "2026-07-07",
	"startDate": "2026-07-07",
	"size": 5,
	"graphType": "DAILY_TREND"
}
```

### Response

`kpiData` **:**  대시보드 상단 kpi 지수

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `totalInspections` | 당일 총 검사 수 | INT |  | X | `100` |
| `yieldRate` | 당일 결함률 | Double |  | X | `95.6` |
| `processStatus` | 공정 진행 상태  | String |  | X | `PENDING`, `RUNNING`, `COMPLETED` |

`summaryData[]` **:** 최근 검사 이력 요약 목록 

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `inspectionId` | 검사 ID | BIGINT |  | X | `1` |
| `finalLabel` | 최종 판정 | VARCHAR(20) |  | X |  `“PASS”`(“PASS”, “REJECT”) |
| `createdAt` | 생성 시각 | TIMESTAMP |  | X | `"2026-07-06T18:00:00"` |

`graphData[]` **:** 요청한 유형의 그래프 데이터 목록

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `label` | 제조사명, 결함 유형, 날짜 | STRING |  | X | `graphType` 별 다름 |
| `value` | 제조사별 결함 건수, 결함별 건수, 날짜별 불량 건수 | INT |  | X | `graphType` 별 다름 |

`graphType` 이 `DEFECT_TYPE`일 때 `label` : 결합 유형 , `value` : 결합 유형별 건수

`graphType` 이 `DAILY_TREND`일 때 `label` : 날짜 , `value` : 날짜별 불량 건수

`graphType` 이 `MANUFACTURE_DEFECTS`일 때 `label` :  제조사명 종류, `value` : 제조사별 불량 건수

**Example**

```jsx
{
  "success": true,
  "message": "대시보드 기본 정보 조회가 완료되었습니다.",
  "data": {
		"kpiData": {
			"totalInspections": 100,
			"yieldRate": 95.6,
			"processStatus": "COMPLETED"
		},
		"summaryData": [
			{
				"inspectionId": 1,
				"finalLabel": "PASS",
				"createdAt": "2026-07-06T18:00:00"
			},
			{
				"inspectionId": 2,
				"finalLabel": "REJECT",
				"createdAt": "2026-07-06T18:00:00"
			}
		],
		"graphData": [
				{ "label": "2026-07-05", "value": 15 },
	      { "label": "2026-07-06", "value": 8 },
	      { "label": "2026-07-07", "value": 5 }
		]
	}
}
```

### Status

| status | response content |  |
| --- | --- | --- |
| **200**  | OK | 서버가 요청을 성공적으로 이해하고 차트 데이터를 반환 |
| **400**  | Bad Request | 프론트엔드에서 보낸 파라미터가 규격에 맞지 않음  |
| **401** | Unauthorized | 헤더에 JWT 토큰이 누락되었거나 만료되어 인증에 실패한 경우 |
| **500** | Internal Server Error | DB 커넥션 오류 등 서버 내부 문제로 조회가 불가능한 경우 |
