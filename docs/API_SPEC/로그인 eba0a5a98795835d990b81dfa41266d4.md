# 로그인

카테고리: login
설명: 아이디와 비밀번호를 통해 로그인
Method: POST
URL: /login
사용자: 유저
상태: 진행 중

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| id | 아이디 | string |  | x | testid123 |
| password | 비밀번호 | string |  | x | testpass123! |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| name | 사용자 이름 | string |  | x | 김성준 |
| role | 사용지 권한 | string |  | x | supervisor |

**Example**

```jsx
{
  "success": true,
  "message": "로그인이 완료되었습니다.",
  "data": {
		"name": "김성준",
		"role": "supervisor"
	}
}
```

### Status

| status | response content |
| --- | --- |
| 200 | OK |
| 400 | Bad Request |
