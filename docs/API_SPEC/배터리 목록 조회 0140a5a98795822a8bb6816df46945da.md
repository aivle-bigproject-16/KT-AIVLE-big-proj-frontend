# 배터리 목록 조회

카테고리: battery
설명: 배터리 목록 조회
Method: GET
URL: /battery
param: page, size 
사용자: 유저
상태: 진행 중

### Request

**Query parameter**

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `page` | 페이지 번호 | Number | 기본값 0 | O | `0` |
| `size` | 페이지 크기 | Number | 기본값 20 | O | `20` |

### Response

`content[]` **:** 배터리 목록

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `inspectionId` | 검사 ID | Number | - | X | `1` |
| `batteryCellId` | 배터리 셀 ID | Number | - | X | `1` |
| `cellSerialNo` | 셀 시리얼 넘버 | String | - | O | `CELL-001` |
| `modelName` | 모델명 | String | - | O | `MODEL-A` |
| `cellType` | 셀 유형 | String | `POUCH`, `CYLINDRICAL`, `PRISMATIC` | O | `POUCH` |
| `latestFinalLabel` | 최근 검사 판정 | String | `PASS`, `REJECT` | O | `REJECT` |
| `latestAnalyzedAt` | 최근 분석 완료 시각 | String | ISO DateTime | O | `2026-07-06T10:30:00` |

`pageable` : 페이징 정보

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `page` | 현재 페이지 | Number | - | X | `0` |
| `pageSize` | 페이지 크기 | Number | - | X | `20` |
| `totalElements` | 전체 개수 | Number | - | X | `120` |
| `totalPages` | 전체 페이지 수 | Number | - | X | `6` |

### Status

| HTTP Status | Code | 설명 |
| --- | --- | --- |
| `200` | `OK` | 조회 성공 |
| `400` | `INVALID_QUERY_PARAMETER` | 잘못된 필터 값 |
| `401` | `UNAUTHORIZED` | 인증 필요 |
| `403` | `FORBIDDEN` | 권한 없음 |
| `500` | `INTERNAL_SERVER_ERROR` | 서버 오류 |
