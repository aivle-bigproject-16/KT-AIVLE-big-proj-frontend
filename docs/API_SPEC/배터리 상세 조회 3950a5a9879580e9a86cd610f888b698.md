# 배터리 상세 조회

카테고리: battery
설명: 개별 배터리 정보 상세 조회
Method: GET
URL: /battery/:batteryCellId
param: :batteryCellId
사용자: 유저
상태: 진행 중

### Request

**Query parameter**

| **파라미터명** | **타입** | **필수 여부** | **기본값** | **설명** |
| --- | --- | --- | --- | --- |
| `:batteryCellId` | Long | Required | - | 조회할 배터리의 id |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `batteryCellId` | 배터리 id | Long | 필수 | X | 1 |
| `cellSerialNo` | 외부에서 식별하는 배터리 cell serial number | String | 필수 | X | "CELL-001" |
| `purchaseId` | 생산처 | String | 선택 | O | "LG" |
| `productId` | 제품 ID | String | 선택 | O | "PRD-001" |
| `modelName` | 배터리 모델명 | String | 선택 | O | "MODEL-A" |
| `cellType` | 셀 유형 | String | 선택 | O | `POUCH`, `CYLINDRICAL`, `PRISMATIC` |
| `manufacturedDate` | 제조일 | String(Date) | 선택 | O | "2026-07-01" |
| `createdAt` | 생성 시각 | String(DateTime) | 필수 | X | "2026-07-07T10:00:00Z" |
| `updatedAt` | 수정 시각 | String(DateTime) | 선택 | O | "2026-07-07T10:00:00Z" |

`inspections[]` : 검사 결과 리스트

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `inspectionId` | 검사 ID | Long | 필수 | X | 101 |
| `finalLabel` | 최종 판정 | String | 필수 | X | `PASS`, `REJECT`, `FAIL` |
| `analyzedAt` | 분석 완료 시각 | String(DateTime) | 필수 | X | "2026-07-07T10:20:00Z" |

`inspections[].image[]` : 검사에 사용된 이미지 목록

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `imageId` | 이미지 ID | Long | 필수 | X | 1001 |
| `imageType` | 이미지 유형 | String | 필수 | X | `CT`, `RGB` |
| `imageUrl` | 이미지 URL | String | 필수 | X | "https://..." |

`inspections[].defectResults[]` : 결함 탐지 결과 목록

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `defectResultId` | 결함 결과 ID | Long | 필수 | X | 501 |
| `label` | 판정 결과 | String | 필수 | X | `REJECT` |
| `imageId` | 해당 이미지 ID | Long | 필수 | X | 1001 |
| `imageType` | 이미지 유형 | String | 필수 | X | `CT`, `RGB` |
| `defectType` | 결함 유형 | String | 필수 | X | `MICRO_DEFECT`, `CRACK`, `SWELLING` |
| `imageUrl` | 결함 이미지 URL | String | 필수 | X | "https://..." |
| `confidence` | 신뢰도 | Float | 필수 | X | 0.9321 |
| `bbox.x` | 바운딩 박스 x 좌표 | Int | 필수 | X | 120 |
| `bbox.y` | 바운딩 박스 y 좌표 | Int | 필수 | X | 80 |
| `bbox.width` | 바운딩 박스 너비 | Int | 필수 | X | 240 |
| `bbox.height` | 바운딩 박스 높이 | Int | 필수 | X | 160 |

`reports[]` : 리포트 리스트

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `reportId` | 리포트 ID | Long | 필수 | X | 1 |
| `inspectionId` | 연관 검사 ID | Long | 필수 | X | 101 |
| `status` | 리포트 상태 | String | 필수 | X | `COMPLETED`, `PENDING`, `FAILED` |
| `title` | 리포트 제목 | String | 필수 | X | "CELL-001 CT 검사 상세 리포트" |
| `createdAt` | 생성 시각 | String(DateTime) | 필수 | X | "2026-07-07T10:40:00Z" |
| `updatedAt` | 수정 시각 | String(DateTime) | 선택 | O | "2026-07-07T10:45:00Z" |

### Status

| HTTP Status | Code | 설명 |
| --- | --- | --- |
| `200` | `OK` | 조회 성공 |
| `400` | `INVALID_PATH_PARAMETER` | 배터리 ID 형식이 잘못됨 |
| `401` | `UNAUTHORIZED` | 인증 필요 |
| `403` | `FORBIDDEN` | 권한 없음 |
| `404` | `BATTERY_NOT_FOUND` | 해당 ID의 배터리가 존재하지 않음 |
| `500` | `INTERNAL_SERVER_ERROR` | 서버 오류 |
