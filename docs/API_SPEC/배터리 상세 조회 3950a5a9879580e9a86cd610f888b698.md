# 배터리 상세 조회

카테고리: battery
설명: 개별 배터리 정보 상세 조회
Method: GET
URL: /battery/:batteryCellId
param: :batteryCellId
사용자: 유저
상태: 진행 중

### Request

**Query parameter**

| **파라미터명** | **타입** | **필수 여부** | **기본값** | **설명** |
| --- | --- | --- | --- | --- |
| `:batteryCellId` | Long | Required | - | 조회할 배터리의 id |

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| `batteryCellId` | 배터리 id | Long | 필수 | X | 1 |
| `cellSerialNo` | 외부에서 식별하는 배터리 cell serial number | String | 필수 | X | “CELL-001” |
| `purchaseId` | 생산처 | String | 선택 | O | “LG” |
| `rgbImageUrl` | rgb이미지url | String |  | O |  |
| `ctImageUrl` | ct 이미지 url | String |  | O |  |
| `productId` | 제품 ID | String | 선택 | O | “PRD-001” |
| `modelName` | 배터리 모델명 | String | 선택 | O | “MODEL-A” |
| `cellType` | 셀 유형 | String | 선택 | O | `POUCH`, `CYLINDRICAL`, `PRISMATIC` |
| `manufacturedDate` | 제조일 | String(Date) | 선택 | O | “2026-07-01” |
| `createdAt` | 생성 시각 | String(DateTime) | 필수 | X | "2026-07-07T10:00:00Z" |
| `updatedAt` | 수정 시각 | String(DateTime) | 선택 | O | "2026-07-07T10:00:00Z" |

### Status

| HTTP Status | Code | 설명 |
| --- | --- | --- |
| `200` | `OK` | 조회 성공 |
| `400` | `INVALID_PATH_PARAMETER` | 배터리 ID 형식이 잘못됨 |
| `401` | `UNAUTHORIZED` | 인증 필요 |
| `403` | `FORBIDDEN` | 권한 없음 |
| `404` | `BATTERY_NOT_FOUND` | 해당 ID의 배터리가 존재하지 않음 |
| `500` | `INTERNAL_SERVER_ERROR` | 서버 오류 |
