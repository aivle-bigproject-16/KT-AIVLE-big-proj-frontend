# 일일 리포트 조회

카테고리: report
설명: 일일 리포트 조회
Method: GET
URL: /reports/daily/:reportId
param: :reportId
사용자: 유저
상태: 진행 중

### Request

**Query parameter**

| **파라미터명** | **타입** | **필수 여부** | **기본값** | **설명** |
| --- | --- | --- | --- | --- |
| `:reportId` | Long | Required | - | 조회할 리포트의 id |

### Response

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| `reportId` | 일일 리포트 id | Long | 필수 | X | 1 |
| `reportDate` | 리포트 기준일  | String(Date) | 필수 | X | "2026-07-06" |
| `status` | 리포트 생성 상태  | String | 필수 | X | `PENDING`, `COMPLETED`, `FAILED` |
| `title` | 리포트 제목 | String | 선택 | O | "상세 리포트" |
| `content` | 리포트 본문 | String | 선택 | O | "금일 검사에서는…" |
| `failureReason` | 실패 사유 | String | 선택 | O | `INCOMPLETE_SET`, `AI_SERVER_ERROR`, `TIMEOUT`, `MALFORMED_RESPONSE`, `PARTIAL_ANALYSIS_FAILURE` |
| `rgbImageUrl` | rgb이미지url | String |  | O |  |
| `ctImageUrl` | ct 이미지 url | String |  | O |  |
| `createdAt` | 생성 시각 | String(DateTime) | 필수 | X | "2026-07-07T10:00:00Z" |
| `updatedAt` | 수정 시각 | String(DateTime) | 선택 | O | "2026-07-07T10:30:00Z" |

`summary` : 보고서 요약

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| `totalCount` | 당일 총 검사 수  | Integer | 선택 | O | `120`  |
| `passCount` | 당일 양품(PASS) 판정 수  | Integer | 선택 | O | `104`  |
| `rejectCount` | 당일 불량(REJECT) 판정 수  | Integer | 선택 | O | `14`  |
| `failedCount` | 당일 시스템 오류/실패 수  | Integer | 선택 | O | `2`  |
| `defects` | 결함 유형별 통계 목록  | Array | 선택 | O | `[]`  |

`summary.defects[]` : 결함 유형별 통계 목록 

| **key** | **설명** | **value 타입** | **옵션** | **Nullable** | **예시** |
| --- | --- | --- | --- | --- | --- |
| `defectType` | 결함 코드/명칭  | String | 선택 | O | `"MICRO_DEFECT"`, `"CRACK"`  |
| `count` | 해당 결함 발생 횟수  | Integer | 선택 | O | `5`, `2`  |

### Status

| **Status** | **명칭** | **발생 시나리오** |
| --- | --- | --- |
| 200 | OK | 일일 리포트 조회 성공 |
| 400 | INVALID_PARAMETER | 리포트 아이디 파라미터 형식이 잘못됨 |
| 401 | UNAUTHORIZED | JWT 토큰이 없거나 만료됨 |
| 404 | REPORT_NOT_FOUND | 해당 일일 리포트가 아직 없음 |
| 500 | INTERNAL_SERVER_ERROR | 리포트 조회 중 서버 오류 |
