# 회원가입

카테고리: auth
설명: 사용자의 정보를 입력하고 계정을 생성
Method: POST
URL: /auth/signup
사용자: 유저
상태: 진행 중

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| id | 아이디 | string |  | x | testid123 |
| password | 비밀번호 | string |  | x | testpass123! |
| name | 사용자 이름 | string |  | x | 김성준 |

### Response

**Example**

```jsx
{
  "success": true,
  "message": "회원가입이 완료되었습니다.",
  "data": {}
}
```

### Status

| status | response content |
| --- | --- |
| 200 | OK |
| 400 | Bad Request |
